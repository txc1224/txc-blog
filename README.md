# txc-blog
 基于rspress实现的blog系统
 [官方文档](https://rspress.dev/zh/)
## 快速开始
1. 初始化项目
``` shell
npm create rspress@latest
```
2. 启动 Dev Server
- 提示
  对于dev命令，你可以通过 `--port` 或 `--host` 参数来指定开发服务的端口号或host，列如 `repress dev --port 8080 --host 0.0.0.0`

``` shell
npm run dev
```
3. 生产环境构建
- 默认情况下，Repress将会把产物打包到 `doc_build` 目录

``` shell
npm run build
```

4. 本地预览产物

``` shell
npm run preview
```
